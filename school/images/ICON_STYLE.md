# Icon Style Guide – Lärande Quiz

## Base prompt template

```
Square app icon, off-white (#f8f8f6) background, [COLOR] thick rounded square outer border,
clean minimal flat line-art, [2–3 SYMBOLS representing topic], single solid [COLOR],
thick uniform strokes, no shadows, no gradients, no neon, no 3D, no textures,
simple geometric shapes, educational school app icon, 1024x1024
```

## Section colors
| Section | Hex       | Usage |
|---------|-----------|-------|
| SO      | #c0306f   | History, geography, democracy |
| NO      | #0095a8   | Science, nature, biology |
| MA      | #7c3aed   | Mathematics |
| SPEL    | #16a34a   | Games |
